利用fabric获取拨号服务器代理

2019-08-13 2 min read # fabric

代理服务器上的web服务没搭建成功，只能采用这种方式了

使用示例

import re
from fabric import Connection

def main():
    c = Connection("用户名@ip", port=0, connect_kwargs={"password": ""})
    ip = re.findall(r'\d+\.\d+\.\d+\.\d+', str(c.run("ifconfig")))[5]
    print(ip)
    c.close()

if __name__ == '__main__':
    main()
