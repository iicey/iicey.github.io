Python使用loguru优雅的打印并存储日志

2022-06-07 1 min read # 日志 # loguru 
from loguru import logger

logger.add(
    "".join([__file__.split(".")[0], ".txt"]),
    level='DEBUG',
    format='{time:YYYY-MM-DD HH:mm:ss} {level} {file}[line:{line}] {message}',
    rotation="10 MB",
    retention='10 days',
    encoding="utf-8"
)
