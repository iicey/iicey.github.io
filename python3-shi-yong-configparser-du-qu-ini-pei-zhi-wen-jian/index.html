Python3使用configparser读取ini配置文件

2019-06-21 1 min read # configparser

新建一个名为config.ini的文件，写入以下内容

[mongodb]
ip=127.0.0.1
port=27017
user=root
pwd=123456

[mysql]
ip=127.0.0.1
port=27017
user=root
pwd=123456

读取配置文件的示例

import configparser

config = configparser.ConfigParser()

config.read('config.ini', encoding='utf-8')
ip = config.get('mongodb', 'ip')
port = int(config.get('mongodb', 'port'))
user = config.get('mongodb', 'user')
pwd = config.get('mongodb', 'pwd')
