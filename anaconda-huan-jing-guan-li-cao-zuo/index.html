Anaconda环境管理操作

2019-07-09 1 min read # Anaconda

Anaconda下载地址，Python下载地址

创建

 conda create -n name python=3.9

列出所有环境

conda env list

激活

source activate name

退出

source deactivate

删除

conda remove -n name --all

shell文件切换conda环境

source /Users/AAAA/anaconda3/etc/profile.d/conda.sh && conda activate env_name
