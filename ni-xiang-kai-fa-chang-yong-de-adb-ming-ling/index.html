逆向开发常用的 adb 命令

2019-09-02 1 min read # adb

真香：https://github.com/mzlogin/awesome-adb

启动 adb 服务

adb start-server

关闭 adb 服务

adb kill-server

查看电脑连接的 Android 设备列表

adb devices

进入 Android shell

adb shell

Android 日志

adb logcat
adb logcat -s keyword

安装 apk

adb install XXX.apk

卸载 apk

adb uninstall XXX.apk

推送文件到 Android 设备

adb push local-path android-path

Android 设备文件传输到电脑

adb pull android-path local-path
