Python3的globals()与locals()

2019-07-01 1 min read # globals # locals

原文链接

在Python中，有两个内建函数，你可能用不到，但是需要掌握它们。

globals() ：以dict的方式存储所有全局变量

def foo():
    print("I am a func")

def bar():
    foo="I am a string"
    foo_dup = globals().get("foo")
    foo_dup()

bar()
# 输出
# I am a func

locals()：以dict的方式存储所有局部变量

other = "test"

def foobar():
    name = "MING"
    gender = "male"
    for key,value in locals().items():
        print(key, "=", value)

foobar()
# 输出
# name = MING
# gender = male
